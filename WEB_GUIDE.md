# Excel Unifier 웹 대시보드 가이드

## 🚀 실행 방법

```bash
./start_app.sh
```

또는

```bash
venv/bin/streamlit run app.py
```

## 🌐 접속

브라우저에서 다음 URL 중 하나로 접속:
- Local: http://localhost:8501
- Network: http://172.24.167.122:8501  
- External: http://112.167.24.169:8501

## 📱 사용 방법

### 1️⃣ 파일 업로드 탭
- **파일 선택**: 드래그 앤 드롭 또는 클릭하여 파일 선택
- **지원 형식**: .xlsx, .xls, .csv
- **다중 선택**: 여러 파일 동시 업로드 가능
- **미리보기**: 각 파일의 데이터를 즉시 확인

### 2️⃣ 데이터 분석 탭
- **키 컬럼 선택**: 중복 제거에 사용할 컬럼 선택
- **유사도 조정**: 사이드바에서 임계값 조정 (0-100)
- **분석 실행**: "분석 및 통합 실행" 버튼 클릭
- **진행 상황**: 실시간 진행률 표시
- **결과 요약**: 처리된 파일 수, 중복 제거 통계 확인

### 3️⃣ 결과 확인 탭
- **데이터 테이블**: 통합된 결과 확인
- **행 수 조정**: 슬라이더로 표시할 행 수 선택
- **다운로드**: Excel 또는 CSV 형식으로 다운로드

### 4️⃣ 통계 탭
- **기본 통계**: 수치형 데이터의 통계 요약
- **결측치 분석**: 컬럼별 결측치 비율 차트
- **파일 기여도**: 각 파일의 데이터 현황 시각화

## ⚙️ 설정 옵션

### 사이드바 설정
- **유사도 임계값**: 
  - 높을수록 (90-100): 엄격한 매칭
  - 중간 (80-90): 균형잡힌 매칭 (기본값: 85)
  - 낮을수록 (70-80): 관대한 매칭

## 💡 팁

1. **대용량 파일**: 파일이 클 경우 처리 시간이 소요될 수 있습니다
2. **키 컬럼**: 이름, 학번 등 고유성이 높은 컬럼을 선택하세요
3. **미리보기**: 통합 전 각 파일의 데이터를 확인하세요
4. **결과 확인**: 통합 후 결과를 꼼꼼히 검토하세요

## 🔒 보안 및 프라이버시

- 모든 처리는 로컬에서 수행됩니다
- 업로드된 파일은 임시 파일로 저장되며 세션 종료 시 삭제됩니다
- 네트워크를 통한 데이터 전송이 없습니다

## 🐛 문제 해결

### 포트 충돌
다른 포트로 실행:
```bash
venv/bin/streamlit run app.py --server.port=8502
```

### 브라우저 자동 열림 비활성화
```bash
venv/bin/streamlit run app.py --server.headless=true
```

### 캐시 초기화
사이드바 메뉴 → "Clear cache"
